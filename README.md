# Handmade Jetpac

A recreation of the ZX Spectrum game Jetpac (1983) written in Handmade Hero style C++. It features Win32 and WebAssembly/WebGL platform layers.

You can play the WebAssembly/WebGL version directly in your browser:  
https://newagebegins.github.io/jetpac/

**Controls:** W, A, D, Space
