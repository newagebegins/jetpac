# Handmade Jetpac

A recreation of a ZX Spectrum game Jetpac (1983) written in Handmade Hero style C++. It has a Win32 and a WebAssembly/WebGL platform layers.

You can play WebAssembly/WebGL version directly in the browser:  
https://newagebegins.github.io/jetpac/

Controls: `W`, `A`, `D`, `Space`
